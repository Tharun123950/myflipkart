# Flipkart Clone Website
## 👩‍💻 Author

Battu Tharun Sai

## 📌 Project Overview

This project is a simple Flipkart-inspired E-Commerce website developed using HTML and CSS. It provides a user-friendly interface for browsing products across multiple categories such as clothes, accessories, footwear, and grocery items.

## 🚀 Features

- Home Page
- Login Page
- About Us Page
- Contact Page
- Product Category Pages
- Responsive Navigation
- User-Friendly Interface
- Simple E-Commerce Website Design

## 📂 Project Files

- index.html – Home Page
- loginpage.html – User Login Page
- aboutpage.html – About Us Page
- contact.html – Contact Page
- clothes.html – Clothing Products
- accessories.html – Accessories Products
- footwear.html – Footwear Products
- grocery.html – Grocery Products

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🎯 Project Objectives

- Learn Frontend Web Development
- Design Multi-Page Websites
- Implement Navigation Between Pages
- Create an E-Commerce Website Layout
- Improve UI Design Skills

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any web browser.
